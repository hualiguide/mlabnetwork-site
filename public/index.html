Agentic Automation for Networks & IoT

We are a student lab building an AI-powered system that monitors, diagnoses, and controls network and IoT devices using agentic automation workflows. Our stack combines GNS3 for network simulation, n8n for workflow orchestration, and modern AI models for intelligent assistants.

The goal: move from manual CLI commands to smart, explainable automation that works in both home-lab setups and enterprise-scale simulations.

About the Project

The AI-Driven Network and IoT Device Management project explores how autonomous agents and workflow engines can support real-world network operations:

  • Monitor routers, switches, firewalls, and IoT nodes.
  • Run diagnostics (pings, traceroutes, config checks) automatically.
  • Summarize results into human-friendly reports via AI.
  • Expose everything through a collaborative web dashboard and chatbot.
